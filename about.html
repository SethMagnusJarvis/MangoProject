RNA-seq Web Tool Analyser

A quick way to analysed RNA-seq data for gene expression analysis

Click here to start

BLAST

The interface allows the user to input fasta files and Blast then against reference genomes for gene prediction.

Gene Expression Analysis

This tool allow the user to carry out Defferential Gene Expression Analysis on all samples.

RESULTS

Results are shown in different format including volcanic plots, hierrarchical clustering and principal component analysis.

This web tool was built by four students as part of a course work for six weeks during MSc Bioinformatics group project
